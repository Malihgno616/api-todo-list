# TODO LIST API

## Clone the repository
```
git clone https://github.com/Malihgno616/api-todo-list.git
```

## Up the docker compose
```
docker-compose up -d
```

## Copy the .env.example
```
cp .env.example .env
```

## Install all dependencies
```
composer install
```

## Migrate databases
```
php artisan migrate
```

## Start the server
```
php artisan serve
```
